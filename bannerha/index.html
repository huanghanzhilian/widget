关闭底部小按钮

new bannerha("#banner5",{ circle: false });

调用方法：

new bannerha(selector,{options});

options参数

参数 默认值 说明
circle true 是否生成底部圆圈按钮
speeds 20 设置缓冲运动速度
pnBtn true 是否生成左右切换按钮
autoPlay true 是否自动轮播
times 3000 设置自动轮播间隔时间
