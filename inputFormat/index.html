货币：

货比格式

new inputFormat('input[name="currency"]');

手机号：

手机格式

new inputFormat('input[name="mobile"]', {
	type: 'mobile'
});

调用方法：

new inputFormat("selector",options,callback(api));

options参数

参数 默认值 说明
type 'currency' 格式化数据类型:'currency'货币,'mobile'手机号
tofixed 2 四舍五入的位数

callback(api)参数

方法 说明
formats(value) 格式化指定值,返回格式化后结果
setValue(value) 设置输入框值
